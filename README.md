# Node.JS Boilerplate with TypeScript

1. TypeScript
2. Docker
3. ES6
4. MongoDB (Mongoose)
5. TSLint
6. TypeScript

## Getting started
1. Install dependencies
```
npm install
```
2. Run production
```
npm run prod
```
2. Or in dev mode (automatic compile typescript)
```
npm run dev
```
3. App is live at:
```
localhost:8080
```
