# Node.JS Boilerplate with TypeScript

1. TypeScript
2. Docker
3. ES6
4. MongoDB (Mongoose)
5. TSLint
6. TypeScript

## Getting started - PROD
1. Install dependencies
```
yarn
```
2. Compile typescript
```
yarn build && yarn start
```
2. Or in dev mode (automatic compile typescript)
```
yarn dev
```
3. App is live at:
```
localhost:8080
```
